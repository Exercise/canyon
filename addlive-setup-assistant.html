Install the HealthFleet plugin.

Step 1: Click the HealthFleet.exe file in the bottom left of the browser window. Click 'Save File' in the popup. In the popup window or popup bar below, click 'Run'. Double-click the HealthFleet.dmg file in the Downloads folder. Double-click the AddLive.dmg file from the Downloads popup.

Step 2: The installer will run, click 'Finish' when it is done. Double-click the HealthFleet.exe file in the Downloads popup. Double-click the installer icon.

Step 3: Please wait, this screen will automatically update. Enjoy! The installer will run, click 'Finish' when it is done. Enjoy! Click "OK" to close the installation dialogue box. The plugin is now installed.
